# model-submit

对于常见表单布局的抽象

## properties
### css

定义model-submit组件的样式

## slots
### content

定义model-submit的内容栏

### bottom

定义model-submit的底部


## example
```
<model-submit css="height:100%">
  <your-content slot="content"></your-content>
  <your-footer slot="bottom"></your-footer>
</model-submit>
```
